# AnyPlace Sport-Health (APSH) - 專業級運動生理監測與自動化評估系統

## 1. 專案簡介
本專案針對民眾在不同運動場所運動後，缺乏專業生理評估與數據追蹤的痛點。系統支援使用者即時登記運動後的「血壓（SBP/DBP）」、「心率（HR）」及「環境氣溫」，並將數據轉換為 **FHIR 國際標準格式**。

系統核心特色在於具備 **CDSS (臨床決策支持系統)** 雛形，能針對「單日多筆紀錄」進行全量掃描，精確定位特定時間點的異常並給予行動指引。

## 2. 核心 FHIR Resources
* **Patient**: 實作使用者註冊與登入驗證，確保生理紀錄與個人身分永久關聯。
* **Observation**: 採用多組件 (Component) 資源，將血壓、心率、氣溫與運動類別封裝於單次量測紀錄。

## 3. 系統功能亮點
* **完整使用者管理 (CRUD)**：
    * 提供新用戶註冊（Create Patient）與舊用戶 ID 驗證登入（Read Patient）。
    * 資料持久化：自動記憶登入狀態，提升使用體驗。
* **智慧化診斷與精準定位**：
    * **單日列表模式**：自動掃描當天所有紀錄，若有多筆量測，系統會精確標註「幾點幾分」出現異常，並提供分級建議。
    * **週趨勢分析**：自動對計算週一至週日平均值，並同步進行「異常全量掃描」。
* **數據修正功能**：支援單筆 Observation 刪除 (Delete)，確保歷史數據的真實性。
* **數據邏輯防錯**：自動限制出生日期與量測時間不可選擇未來日期。

## 4. 如何執行 (How to Run)
1. 使用 VS Code 開啟資料夾，確保 `fhir.html` 與 `script.js` 處於同目錄。
2. 使用 **Live Server** 開啟 `fhir.html`。
3. 進行新用戶註冊後，即可開始上傳與分析數據。
